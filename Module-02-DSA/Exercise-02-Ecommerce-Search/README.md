# Exercise 02 - E-commerce Search Function

## Objective
Implement Linear Search and Binary Search for product searching.

## Files
- Product.java
- SearchAlgorithms.java
- SearchTest.java

## Algorithms
- Linear Search
- Binary Search

## Time Complexity

Linear Search
Best Case: O(1)
Average Case: O(n)
Worst Case: O(n)

Binary Search
Best Case: O(1)
Average Case: O(log n)
Worst Case: O(log n)

## Learning Outcome

Understand searching techniques and compare Linear Search with Binary Search.
