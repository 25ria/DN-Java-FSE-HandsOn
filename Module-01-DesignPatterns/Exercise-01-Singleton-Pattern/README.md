# Exercise 1 - Singleton Pattern

## Objective

Implement the Singleton Design Pattern by creating a Logger class that ensures only one instance exists throughout the application.

## Files

- Logger.java
- SingletonTest.java

## Concepts Used

- Singleton Design Pattern
- Classes and Objects
- Static Variables
- Private Constructor
- Static Methods

## Output

```
Logger instance created.
LOG: First message
LOG: Second message
Both logger objects are the same instance.
```

## Learning Outcome

Learned how to implement the Singleton Design Pattern in Java using a private constructor and a static instance.
